MEAN CRUD
===================
MongoDB / Express / AngularJS / Node.js

Technologies
-------------------
- Node.js (Server platform)
- Express (Backend JS framework)
- MongoDB (NoSQL database)
- AngularJS (Frontend JS framework)
- Grunt (Task manager)
- SASS (CSS preprocessor)
